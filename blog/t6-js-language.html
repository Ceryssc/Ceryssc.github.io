In the JavaScript language, describe all the use cases for each of:

parentheses ( ) - Used to contain a list of parameters passed to functions. Used for grouping expressions to enforce the Order of Operations. Used for containing expressions e.g. in an ‘if’ statement. Used to call functions.

brackets [ ] - Used for accessing the properties of an Object and the elements of an Array

braces { } - Used to declare Objects. Also used to enclose blocks of code in function definitions, conditional blocks:

  if (10 > 5) {
                  var outcome = "if block";
            }

single quotes '' or double quotes " " - Used for writing strings. Neither style has special semantics over the other. However can be used strategically by the programmer the help understanding reading of the code. The one thing that should be noted is that JSON (JavaScript Object Notation) only uses double quotes: 

 var myObject = {
            "element" : "value"
            };
