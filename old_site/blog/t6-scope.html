What 'scope' is, and how it works in JavaScript.

Scope determines the accessibility/visibility of variables, functions and objects in areas of code.

Global Scope - If a variable is defined outside of a function, it has global scope. There is only one global scope throughout a Javascript document.

Local scope - If a variable is dinned inside of a function, it has local scope. These variables are only recognised inside of the function, they are are bound to the function and are not accessible in other functions. This also means variables with the same name can be used in different functions.
